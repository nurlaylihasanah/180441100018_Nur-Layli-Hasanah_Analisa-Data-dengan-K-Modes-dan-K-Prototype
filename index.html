    Analisa Data dengan K-Modes

    Sumber data : https://data.world/uci/flags/workspace/file?filename=flag.data.csv

    Tipe Data Categorical

    Source code :

    import numpy as np
from kmodes.kmodes import KModes

x = np.genfromtxt('flag.data.csv', dtype=int, delimiter=',')[:, :-1]

kmodes_huang = KModes(n_clusters=4, init='Huang', verbose=1)
kmodes_huang.fit(x)

print('k-modes (flag.data.csv) centroid:')
print(kmodes_huang.cluster_centroids_)
print('Final training cost: {}'.format(kmodes_huang.cost_))
print('Training Interations: {}'.format(kmodes_huang.n_iter_))

    Analisa Data dengan K-Prototype

    Sumber data : https://data.world/uci/zoo/workspace/file?filename=zoo.data.csv

    Tipe Data Campuran (Numerical & Categorical)

    Source code :

    import numpy as np
from kmodes.kprototypes import KPrototypes

syms = np.genfromtxt('zoo.data.csv', dtype=str, delimiter=',')[:, 0]
X = np.genfromtxt('zoo.data.csv', dtype=object, delimiter=',')[:, 1:]
X[:, 0] = X[:, 0].astype(float)

kproto = KPrototypes(n_clusters=2, init='Cao', verbose=2)
clusters = kproto.fit_predict(X, categorical=[9, 2])

print(kproto.cluster_centroids_)
print(kproto.cost_)
print(kproto.n_iter_)

for s, c in zip(syms, clusters):
    print("Symbol: {}, cluster:{}".format(s, c))
